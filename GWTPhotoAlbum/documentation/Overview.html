GWTPhotoAlbum is a free (Apache 2.0 License) AJAX library for an online photo gallery and slide show.

Project Home: code.google.com/p/gwtphotoalbum/

Features:

  1. Uses the full browser window size, i.e. automatic resizing of images
  2. Highly configurable, i.e. different layouts for the slide show, embeddable in html pages
  3. Supports the display of image captions
  4. Supports all majow browsers: Firefox 2.0 and higher, Opera 9.5 and higher, Internet Explorer 7 and higher, Safari 3 and higher, Google Chrome; (unfortunately it does not work with the KDE-Konqueror browser :( )
